Privacy Policy

Last updated: March 30, 2026

Overview

Moodle Content Extractor for NotebookLM is a local-first browser extension. Its purpose is to capture content from Moodle lesson and activity pages while capture mode is enabled by the user and access has been granted through explicit user action, organize that content locally, and export it as Markdown or backup files when the user requests it.

What data the extension handles

The extension may process the following data inside the user's browser:

  • Moodle page content visible to the user.
  • Course names, module names, captured page URLs, timestamps, and extracted text.
  • Links to embedded videos and downloadable resources.
  • User settings such as the configured Moodle base URL.
  • Local backup and recovery snapshots created by the extension.

Why URL and timestamp metadata is stored

Captured page URLs and timestamps are stored locally to organize modules, avoid duplicate captures, preserve capture order, and keep exports and backup/restore data consistent.

How data is stored

Data is stored locally in the browser using Chrome extension storage. The extension also allows the user to export data as Markdown or JSON backup files.

What data is not collected

  • No analytics are sent by this extension.
  • No personal data is sold.
  • No user account is required.
  • No external server is required for normal use of the extension.

When data leaves the browser

The extension only causes data to leave the browser when the user explicitly exports a file, imports a backup, or uploads exported content to another product such as NotebookLM.

User control

  • Users manually choose when to start and stop capture mode.
  • Users trigger capture access through explicit interaction with the extension UI.
  • Users can clear stored data.
  • Users can export backups and restore them manually.
