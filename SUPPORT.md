# Support

## What to use for what

- Use GitHub Issues for bugs and feature requests.
- Use Discussions, if enabled later, for general questions and ideas.
- Use private contact with Fernando Paladini at fnpaladini@gmail.com for security-sensitive issues.

## Before opening an issue

- Confirm your Moodle base URL is configured correctly.
- Reload the extension in `chrome://extensions`.
- Reproduce the issue on a single course if possible.
- Check whether the problem affects capture, export, or backup separately.
